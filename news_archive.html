 News Archive
December 14, 2004: AJDT 1.2.0M2 released. See the New and Noteworthy page for 1.2.0M2 for an overview of the release, or go straight to the download page. See also the latest release notes for this release.
November 9, 2004: "Easy Install" AspectJ and AJDT distributions available. Russ Miles has packaged together integrated installers for Eclipse, AJDT, and AspectJ on the Windows and Mac platforms. So now you can install a fully-configured AspectJ development environment in one single step. Also look out for his forthcoming book, the AspectJ Cookbook.
November 2, 2004: New AJDT demos. See the demos page for several animated demonstrations including one showing refactoring of crosscutting concerns in an existing project, and another showing use of aspect libraries.

September 22, 2004: AJDT article published on developerWorks: Develop aspect-oriented Java applications with Eclipse and AJDT.
August 13, 2004: AJDT 1.1.12 released. See the New and Noteworthy page for 1.1.12 for an overview of the release, or go straight to the download page.
September 22, 2003: AJDT 1.1.4 released, with full support for AspectJ 1.1.1. See the New and Noteworthy page for more details. One new contribution for this release is the User's Guide. Installation instructions can be found on our download page.
