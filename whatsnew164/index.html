AJDT 1.6.4 - New and Noteworthy

he AJDT team is pleased to announce the release of AJDT 1.6.4. This release includes over 50 bug fixes. It also includes AspectJ version 1.6.4.20090304172355 which has an additional 43 bugs fixed since 1.6.3. This version of AspectJ is available for download separately from AJDT. See the downloads page for installation instructions.

Thanks to the AJDT and AspectJ community who provided valuable help through testing, bug reporting, feature requests, providing test cases and patches, and being generally supportive of newcomers.
