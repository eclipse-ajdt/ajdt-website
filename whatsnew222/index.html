AJDT 2.2.2 - New and Noteworthy

See bugzilla for a list of bugs addressed in this release.
Overview AJDT 2.2.2 is a service refresh and contains AspectJ 1.7.2 as well as bug fixes. See the downloads page for information on how to install this release.
Build time improvements Because of some extra caching of file system locations for source files, we have been able to significantly trim the build times of AspectJ projects. This fix particularly improves build times for projects with large numbers of files. For more information, see bug 394569.
Eclipse 4.3 support Snapshot builds of AJDT for Eclipse 4.3 (Kepler) are now available. See the downloads page for information. And use the following udpate site to install:
  • http://download.eclipse.org/tools/ajdt/43/dev/update
    AspectJ 1.7.2 AJDT 2.2.2 includes a AspectJ with version 1.7.2.20130214224500. See the release notes for 1.7.1 for more information. This is a link to the actual AspectJ version available in this AJDT release.
