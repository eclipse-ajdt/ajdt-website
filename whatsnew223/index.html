AJDT 2.2.3 - New and Noteworthy

Overview AJDT 2.2.3 is a service refresh and contains AspectJ 1.7.3 as well as bug fixes. See the downloads page for information on how to install this release.
Eclipse 4.3 support AJDT 2.2.3 is the first release with Eclipse 4.3 (Kepler) support. There is no Eclipse 3.7 (Indigo) versuion of this release. See the downloads page for information. And use the following udpate site to install:
  • Eclipse 4.3 Kepler http://download.eclipse.org/tools/ajdt/43/update
  • Eclipse 4.2 and 3.8 Juno http://download.eclipse.org/tools/ajdt/42/update
    Restructuring of AspectJ bundles The AspectJ bundles (org.aspectj.runtime, org.aspectj.weaver, org.aspectj.ajde) are have been restructured to be proper OSGi bundles. This has the following advantages:
    • Smaller install size by about 6 Mb since sources and unused documentation have been removed.
    • Separately installable AspectJ feature. It is now possible to consume AspectJ without installing the rest of AJDT.
    • Separately installable source bundles for AspectJ.
    AspectJ 1.7.3 AJDT 2.2.2 includes a AspectJ with version 1.7.3.20130613144500. See the release notes for 1.7.3 for more information. This is a link to the actual AspectJ version available in this AJDT release.
